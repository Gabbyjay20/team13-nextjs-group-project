# Team 13 Design Theme

## Theme Name
Modern Blue Dashboard

## Purpose
The application will use a clean, modern dashboard interface that is easy to navigate, responsive, and accessible on both desktop and mobile devices.

## Color Palette

| Purpose | Color |
|---------|-------|
| Primary | #2563EB (Blue) |
| Secondary | #FFFFFF (White) |
| Background | #F8FAFC (Light Gray) |
| Accent | #0EA5E9 (Sky Blue) |
| Text | #1F2937 (Dark Gray) |

## Typography

- Primary Font: Inter
- Secondary Font: Sans-serif

## Layout

- Responsive design
- Left sidebar navigation
- Dashboard cards
- Clean spacing
- Mobile-friendly interface
