# Team 13 - Next.js Group Project

## Project Overview

This repository contains the collaborative project for Team 13 in the BYU-Idaho Next.js course.

The objective is to build a modern full-stack web application using Next.js while practicing GitHub collaboration, project planning, and software engineering principles.
